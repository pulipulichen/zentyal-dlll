Domain Name Format Help

Format

Domain Name Section Format:

"[type]-[name]-[year].[upper-domain-name]"

For example:

"test-zentyal-2013.dlll.nccu.edu.tw"

Section

[type]

Usage.

  • exp (experiment): for 1 semester. 
    exp-kals-2012.dlll.nccu.edu.tw
  • test: discard in anytime.
    test-pudding-2013.dlll.nccu.edu.tw
  • pc (personal computer): for 3 years.
    pc-pudding-2013.dlll.nccu.edu.tw
  • teach: for 1 semester.
    teach-dspace-dlll-2013.dlll.nccu.edu.tw
  • demo: for a long time
    demo-kals-2013.dlll.nccu.edu.tw
  • public: for a long time.
    public-kals-2013.dlll.nccu.edu.tw
  • (non-type): equal to type public, for a long time.
    reading-club-2013.dlll.nccu.edu.tw

[name]

Project name or userâ€™s name.

  • kals: a project name.
    demo-kals-2009.dlll.nccu.edu.tw
  • dspace-dlll: a project name.
    demo-dspace-dlll-2013.nccu.edu.tw
  • dspace-dlll-01: a project name for group 01.
    teach-dspace-dlll-01-2013.dlll.nccu.edu.tw
  • pudding: a user name.
    pc-pudding-2013.dlll.nccu.edu.tw
  • red: a user name.
    pc-red-2012.dlll.nccu.edu.tw

[year]

Created year.

  • 2012: Created from 2012.
    pc-red-2012.dlll.nccu.edu.tw
  • 2013: Created from 2013.
    demo-dspace-dlll-2013.nccu.edu.tw

[upper-domain-name]

Domain name which DNS could control.

  • dlll.nccu.edu.tw: for DLLL LAB use.
    teach-dspace-dlll-01-2013.dlll.nccu.edu.tw
  • lias.nccu.edu.tw: for LIAS use.
    teach-koha-2013.lias.nccu.edu.tw
